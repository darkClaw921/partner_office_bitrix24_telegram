Система работы с партнерами и клиентами в Bitrix24.

## Структура

- `partner` - бот для работы с партнерами (личный кабинет партнера)
- `users` - бот для работы с клиентами (пришедшие от партнёра клиенты)
- `utm_partner_binding` - сервис для привязки партнеров к сделкам и лидам (поиск партнера по коду из UTM-метки)
- `detail_deal_for_partner` - сервис для встраивания в элемент сделки (отображение сделок/лидов пришедших от партнёра)

## Установка

```bash
uv sync
```
